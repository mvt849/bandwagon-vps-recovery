# 搬瓦工忘记续费中断怎么办？VPS 还能救回来吗：账单状态判断、服务恢复步骤、以及如何彻底避免下次再忘

这时候大多数人的反应是懵的。数据还在吗？还能续费吗？过了多少天了？

先深呼吸一下。大概率还有机会救。

所以第一步不是慌，是去后台查账单状态。

## 先搞清楚：搬瓦工的续费机制是怎么运作的

搬瓦工不会自动从你的银行卡或支付宝扣款。官方明确说了"we never charge your payment method automatically"——所有付款都是用户主动发起的，没有例外。

所谓"自动续费"，只有一种情况：你提前往账户充了余额（Credit），账单出来的时候系统自动用余额扣。余额不够，还是得你手动付。

账单出来的时间，目前已经改为**到期日前 7 天生成账单**，加上到期后还有 7 天宽限期，实际上你有 **14 天的付款窗口**。7 天宽限期内，VPS 照常运行，不中断。7 天过了还没付，才会被暂停（Suspended）。暂停之后账单通常还有效，付了就能恢复。再拖更久，账单被系统取消，VPS 和数据一起删除，不可恢复。

这是整个时间线，没什么复杂的：

1. **到期前 7 天**：账单生成，邮件通知，账户余额充足则自动扣款
2. **到期后 0-7 天（宽限期）**：VPS 正常运行，手动付款即可续费
3. **到期后第 7 天**：VPS 被暂停（Suspended），但账单通常还在
4. **账单被系统取消后**：VPS 彻底删除，数据归零，无法恢复

## 你现在到了哪一步？对号入座

### 情况 A：VPS 还在运行，只是出了续费通知

最轻松的情况。直接登录后台，进 Billing → My Invoices，找到 Unpaid 账单，点进去付款就行。支付成功，服务周期自动延长，一分钟搞定。

### 情况 B：VPS 显示 Suspended，但账单还在

也还救得了。操作和情况 A 一样——My Invoices 找 Unpaid 账单，付款，VPS 立刻恢复。不需要联系客服，不需要等待，付完就好。

顺便检查一下邮箱垃圾箱，续费提醒邮件有时候会被误判成垃圾邮件。标题大概是"Invoice #XXXXX Generated"，找到它、加入白名单，下次就不会漏了。

### 情况 C：进 My Invoices，账单状态是 Cancelled

这个情况比较麻烦。账单一旦被系统取消，搬瓦工的策略是不可恢复——VPS 连同所有数据已经被删除了。联系客服通常也没用，系统层面就是这样设计的。

此时唯一的路是重新购买一台 VPS，从头来过。

## 操作步骤：找到账单并完成付款

前提：你的 VPS 续费开关（Renewal Invoices）必须是 **ON** 状态。如果你之前手动关过，账单根本不会生成，到期就直接没了，也救不了。

**手动付款流程（4 步）：**

1. 登录搬瓦工后台，进入 Billing → My Invoices
2. 找到状态为 **Unpaid** 的续费账单
3. 点击 View Invoice，进入账单详情页
4. 点击 **Pay Now**，选择支付宝 / PayPal / 信用卡完成付款

付款成功后，VPS 状态会立刻从 Suspended 切换回 Active，SSH 重新连上。

如果在 My Invoices 里找不到 Unpaid 账单，有几种可能：
- 账单还没到出账时间（到期日前 7 天才出账）
- 账户余额已经自动支付过了（Billing 首页会显示余额）
- 账单已被取消（情况 C）

## 邮件没收到怎么回事？

搬瓦工忘记续费有一半是邮件漏了导致的，不是没发，是没收到。常见原因：

- 注册邮箱填错了（尤其是当年随便注册时）
- 邮件被丢进了垃圾箱（Gmail、QQ 邮箱都有这毛病）
- 注册的是临时邮箱，早就失效了

解决办法不复杂。登录后台，检查右上角账户信息里的邮箱是不是自己在用的那个，不对就改过来。然后把搬瓦工的发件域加进邮件白名单，下次就不会再漏了。

充值入口：后台 → Billing → Add Funds，支持 PayPal 和信用卡（注意：Add Funds 不支持支付宝，但续费账单支付本身是支持支付宝的）。

## 续费之后：设一个提醒，或者充上余额

付完这次续费，顺手做一件事：要么在手机日历里设一个提前两周的提醒，要么直接给账户充上足额余额，让系统下次自动扣。

手动续费方式：账单出来了付就行，支付宝、PayPal、信用卡都支持，支付宝是其中最方便的，扫码秒付。

## 常见问题

**Q：搬瓦工会不会自动扣我银行卡？**

不会。搬瓦工不存储任何支付信息，也不会主动扣你银行卡或支付宝。所有扣款都是用户主动付款，或者用账户余额自动扣——余额是你提前自己充进去的。

**Q：账单出来了但我当天没看到邮件，过了两天才发现，还来得及吗？**

来得及。到期后有 7 天宽限期，宽限期内 VPS 正常运行，账单有效，随时付随时恢复。就算宽限期过了 VPS 被暂停，账单通常也还在，付了就能恢复。只有账单被系统彻底取消才救不了。

**Q：我的 VPS 续费之后价格会涨吗？**

续费价格一般按当前官方售价计算。如果你是早年买的老方案，价格可能比现在低很多，这种情况下续费延续老价格——但如果你修改过续费周期（比如月付改年付），价格会按现价重新计算，要注意。

**Q：我之前手动关了续费开关，现在想开回来，怎么操作？**

后台 → Services → My Services → 找到对应 VPS → 齿轮图标 → Manage Renewal → 把 Renewal Invoices 切换回 ON → Save Changes。开关打开之后，到期时就会正常生成续费账单了。

**Q：搬瓦工忘记续费之后数据完全找不回来了吗？有没有备份？**

如果账单已经被取消、VPS 被删除，官方没有恢复途径。搬瓦工提供免费自动备份功能（在 KiwiVM 面板里可以看到），但这个备份在 VPS 被删除的同时也会一起清掉，不是独立存储的。所以重要数据一定要在续费之前定期自己拉出来。
